Server Rules

Our server policy.

What you can expect from us:
We are all in the same boat, so you can expect active GMs without the abuse of power. GM powers will only be used for resolving issues and enforcing the rules. We are open to suggestions, if you think you have good idea which may improve player experience on the server - we are all ears!

Server size:
The more the merrier! We have initially opened the server with 16 available player slots. Once we can gauge how interested people are in staying on and actively playing, and find that it’s popular, we will increase the player slots to 32, and then 64 if necessary.

Player Abuse:
We want the server to be a nice place, and we want people to enjoy the server. The gameplay should be as natural as possible, whilst still being enjoyable. You might encounter a few villains, bandits and beggars, but we won’t accept outright, hurtful abuse or bullying outside of RP.
